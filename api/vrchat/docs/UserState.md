# UserState

* \"online\" User is online in VRChat * \"active\" User is online, but not in VRChat * \"offline\" User is offline  Always offline when returned through `getCurrentUser` (/auth/user).

## Enum

* `Offline` (value: `'offline'`)

* `Active` (value: `'active'`)

* `Online` (value: `'online'`)
